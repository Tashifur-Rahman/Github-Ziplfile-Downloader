GHZIP

Download any public GitHub repository as a ZIP — directly, without leaving this page.

About

GHZIP is a lightweight, self-hostable tool that lets you download any public GitHub repository as a ZIP archive. Paste the repository URL, enter your access key, and the server fetches the archive directly from GitHub's CDN and streams it to your browser — the source URL is never exposed to the client.

The tool is key-gated, meaning only users with a valid access key can trigger downloads. This makes it suitable for controlled release environments where you want to share a project privately without making the repository public.
